info_outline Actividad: Instalación del ambiente de programación.

Realizarás la instalación del ambiente de programación que estaremos usando a lo largo del semestre.

check Objetivos de aprendizaje
  • Instalación del ambiente de desarrollo.
list Instrucciones
  • Escoge la opción que prefieras:

    A) (Principiante) Si es tu primera vez programando baja e instala Thonny que es un programa que tiene un IDE (editor) y un compilador para principiantes en todos los sistemas operativos. Posteriormente durante tu se espera que aprendas a compilar desde consola y migres hacia otros IDEs más avanzados.

    B) (Avanzado Win) Si ya tienes experiencia y quieres usar el compilador directamente en windows: este video tutorial intalando Python 3 en win 10 donde se explica:
    1) Come se baja e instala Python 3 que es el compilador independiente.
    2) Como se baja e instala un IDE (Integrated Development Environment) que es editor de texto para programar. en este caso notepad++ para windows o sublime para mac o linux.

    C) (Avanzado Mac) Si ya tienes experiencia y quieres el compilador en mac sigue el siuiente tutorial Installing Python 3 on Mac OS X para mayo documentación técnica puedes consultar python-396 .

    D) (Remoto) Si no tienes una computadora propia o prefieres programar en tu navegador web, entonces crea una cuenta en Repl.it que es una página de internet que ya tiene instalado el compilador y el editor. Lee la documentación para que entiendas como se guardan los archivos en línea para que no pierdas tus avances.
